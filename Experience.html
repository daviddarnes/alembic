显微镜实验平台搭建

显微镜是观察马氏体相变的有效办法之一，比起原子力显微镜（AFM）可以看到材料相变的全貌，速度快，但并不能看到更微观的结构。我们研究组用的显微镜是Nikon Ni-U显微镜，并打算利用SDK对显微镜的照相机进行二次开发，提取原始的数据，并对原始的数据进行处理，以达到能用颜色代表高度梯度的目的。

X光衍射图谱模拟

根据布拉格晶格衍射的原理，考虑结构因子，温度因子，衍射角度等因素，利用MATLAB便可以模拟出X光对某一特定晶体材料的衍射图谱。详情请参考XRD Simulation。

Building up the microscope system to observe the Martensite transformation

Microscope is a efficient way to observe the Martensite transformation. It can catch the whole picture of the phase transformation quite fast. But it cannot reach the resolution as the AFM. Our research group is using Nikon's Ni-U microscope to take the picture of the Martensite phase transformation. We are going to use SDK package to develop the CCD, get the raw data and make the CCD being suitable to our target.

XRD simulation

According to Bragg's law, considering the structure factor, temperature factor, diffraction angle and other factors, we can use MATLAB as our tool to code a program to simulate the XRD result of a particular crystal material. Please refer to XRD Simulation.
